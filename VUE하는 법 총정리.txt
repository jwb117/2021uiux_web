[번외] npm install :  node_modules 폴더 재설치할때 입력

[번외] git clone 경로입력(https://github.com/경로명(개인마다다름) : 내려받기(복제, 풀다운)

[번외] 처음에 node.js 사이트 들어가서 인스톨 파일을 다운받아서(무조건 인터넷 접속되야함) 설치 후 진행

[0] cmd 에서 node 먼저 설치하기

[1] node.js install : 노드.js 처음 설치할때 입력(cmd-node 모드에서)

[2] npm install -g @vue/cli // 뷰 cli까는 거(노드 cmd에서 치기)

[3] vue --version : 뷰 버젼 확인

[4] node -v : 노드 버젼 확인

[5] vue create 작명 : 프로젝트(폴더) 만들기 명령어

[6] github 레포지터리 생성하기(github 들어가서 new 눌러서 만들기)

[7] remote연결하기 (순서대로 한줄씩 입력하기)
git remote add origin https://github.com/경로명(개인마다다름)
git branch -M main
git push -u origin main 입력하면 푸시까지 다되어서 레파지토리 완성됨)

[8] code . : 비쥬얼 스튜디오 코드 실행(cmd창에서)

[9] npm run serve : 서버실행 명령어(라이브서버로 브라우저로 실시간 볼 수 있음)
